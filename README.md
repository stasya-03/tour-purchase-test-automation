## Описание проекта
Проект автоматизирует процесс тестирования формы оплаты тура картой.
В ходе выполнения были реализованы автотесты на основе Selenide и JUnit5.
Результаты тестов отображаются в Allure Report.

## Требования
Перед запуском необходимо установить и запустить следующие компоненты:
- Java 11+
- Docker (для запуска контейнера с базой данных и приложением)
- Gradle
- Google Chrome

## Установка и запуск
1. Клонировать репозиторий:

   git clone <URL>
   cd <project-folder>
   

2. Запустить контейнеры приложения и базы данных:
   
   docker-compose up
   

3. Выполнить команду для запуска тестов:
   
   ./gradlew clean test
   

4. Для формирования отчета Allure выполнить:
   
   ./gradlew allureReport
   ./gradlew allureServe
   

## Используемые технологии
- Java 11
- Gradle
- JUnit5
- Selenide
- Allure
- Docker
- MySQL
